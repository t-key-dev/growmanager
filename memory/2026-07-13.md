## GrowManager App (Cannabis Grow-Zelt-Verwaltung)

- User wollte mobile App zum Managen von Cannabis-Growzelten: Grow-Pläne (Gießmenge, pH, Dünger, Lampenstärke/-abstand, Luftfeuchte, Temperatur), Kalender mit Auto-Terminen (Gießen/Düngen), Zelte anlegen + Pflanzen zuordnen, integrierter VPD-Rechner. User kann nicht programmieren, volle Umsetzung an mir.
- Gebaut als React + TypeScript + Vite PWA, lokale Datenhaltung via Dexie (IndexedDB), kein Backend/Login nötig.
